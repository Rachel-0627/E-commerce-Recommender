# 第一阶段产品需求

## 产品定位

面向电商平台的个性化商品推荐与 AI 推荐解释系统。系统不仅展示推荐商品，还要用自然语言解释推荐原因，提高用户对推荐结果的理解和信任。

## MVP 目标

第一阶段先完成最小闭环：

```text
用户请求推荐
  -> 系统读取用户画像和商品信息
  -> 生成推荐商品列表
  -> 生成推荐理由标签
  -> 返回自然语言解释
  -> 预留用户反馈入口
```

## 核心用户场景

### 首页推荐

用户进入首页后，系统展示个性化推荐商品，并显示一句简短推荐解释。

### 商品详情页推荐

用户浏览某个商品时，系统推荐相似商品或搭配商品，并解释推荐原因。

### 冷启动推荐

当用户没有足够行为数据时，系统使用热门商品、高评分商品、类目偏好进行兜底推荐。

## 第一阶段功能范围

- 用户画像读取
- 商品数据读取
- 模拟推荐接口
- 推荐理由标签生成
- 模板化 AI 解释
- 推荐结果统一返回格式
- 后续行为埋点和解释反馈接口预留

## 暂不实现

- 真实登录系统
- 在线支付和订单链路
- 大规模实时特征平台
- 深度学习排序模型
- LLM 微调
- A/B 测试平台

## 成功标准

- 推荐 API 可以稳定返回商品列表
- 每个推荐商品带有 reason_tags 和 explanation
- 数据结构支持后续接入数据库、推荐算法和 LLM
- 项目可以本地启动并通过 API 文档调试
